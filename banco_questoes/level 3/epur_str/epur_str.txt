Assignment name  : epur_str
Expected files   : epur_str.c
Allowed functions: write
--------------------------------------------------------------------------------

Escreva um programa que receba uma string e exiba essa 
string com exatamente um espaço entre as palavras, 
sem espaços ou tabs nem no início nem no fim, seguido 
por um \n.

Uma "palavra" é definida como uma parte de uma string 
delimitada por espaços/tabs, ou pelo início/fim da string.

Se o número de argumentos não for 1, ou se não houver 
palavras para exibir, o programa exibe \n.

Exemplo:

$> ./epur_str "See? It's easy to print the same thing" | cat -e
See? It's easy to print the same thing$
$> ./epur_str " this        time it      will     be    more complex  . " | cat -e
this time it will be more complex .$
$> ./epur_str "No S*** Sherlock..." "nAw S*** ShErLaWQ..." | cat -e
$
$> ./epur_str "" | cat -e
$
$>
